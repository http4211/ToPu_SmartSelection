# ToPu SmartSelection

**ToPu SmartSelection** は、Blender の選択管理と Local View を使いやすくするアドオンです。  
オブジェクト、コレクション、メッシュ要素の選択を保存・呼び出ししながら、選択範囲だけをすばやく表示できます。

## Features

- オブジェクト選択セットの保存・呼び出し
- コレクション単位の選択補助
- メッシュ選択セットの保存・呼び出し
- 選択範囲を階層的に表示する **TP Smart Local View**
- Mesh / Curve / Surface / Armature / Metaball の編集モード対応
- デフォルトショートカット対応
  - `Numpad /`：TP Smart Local View
  - `Ctrl + J`：選択系操作

## Requirements

- Blender 4.0 or later

## Installation

1. Blender を開く
2. `Edit > Preferences > Add-ons` を開く
3. `Install from Disk...` から zip ファイルを選択
4. **ToPu SmartSelection** を有効化

## Usage

アドオンを有効化すると、選択セット管理と TP Smart Local View が使用できます。  
よく使うオブジェクト、メッシュ要素、コレクションを登録しておくことで、複雑なシーンでも素早く選択・表示を切り替えられます。
